<p align="right"><code>// 04 — active projects</code></p>

**📊 ML Metrics Calculator**

Aplicação em Python + Streamlit para avaliação de modelos de Machine
Learning: métricas de classificação e regressão, gráficos
interativos (matriz de confusão, curva ROC, precision-recall) e
explicações sobre quando uma métrica pode enganar — como acurácia
alta mascarando recall baixo em dados desbalanceados.

`Python` `scikit-learn` `Streamlit` `pytest`
&nbsp;·&nbsp;
[repositório](https://github.com/bruno-dsn/ml-metrics-calculator)
&nbsp;·&nbsp;
[app](https://ml-metrics-calculator.streamlit.app)

<br>

**📈 Painel de Preços de Ações (B3)**

Dashboard interativo em Streamlit para visualização de ativos
negociados na B3, com dados históricos via `yfinance`. Foco em
análise exploratória de dados financeiros e visualização interativa
de séries de preços.

`Python` `Streamlit` `Pandas` `yfinance` `Matplotlib`
&nbsp;·&nbsp;
[repositório](https://github.com/bruno-dsn/painel-precos-acoes)
