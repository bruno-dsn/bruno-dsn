<p align="right"><code>░░░ 01 · SOBRE MIM</code></p>

Trabalho na camada onde dado bruto vira decisão: estatística para
entender o problema, modelos para generalizar o padrão, e uma
interface para que o resultado saia da minha máquina e chegue em
alguém. Cursando atualmente a Pós-Tech em AI Scientist (FIAP + Alura),
com o eixo em Machine Learning, Deep Learning, NLP e IA Generativa.

Tenho formação prévia em Análise de Dados (EBAC) e venho de uma base
técnica em redes de computadores — o que faz eu olhar para um modelo
não só pela acurácia, mas por como ele se comporta quando algo sai do
esperado.
